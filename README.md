# MahaPoly Search

Maharashtra DTE Polytechnic CAP cutoff explorer. Enter your merit % and category
to see which colleges and branches you qualify for, across 418 institutes and
~77,000 real CAP cutoff records.

## Run locally
npm install
npm run dev

## Deploy
Push this folder to a GitHub repo, then import it in Vercel. No configuration needed —
Vercel auto-detects Next.js.
